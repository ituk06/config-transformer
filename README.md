# Конфигурационный трансформер (Вариант 30)

Инструмент командной строки для преобразования учебного конфигурационного языка в TOML.

## Возможности языка (Вариант 30)

- **Числа**: `[+-]?\d+` (целые числа со знаком)
- **Строки**: `@"текст"` (в двойных кавычках с префиксом @)
- **Массивы**: `{значение, значение, ...}` (с запятыми)
- **Словари**: `table( ключ => значение, ... )`
- **Константы**: `var имя := значение;`
- **Выражения**: `|выражение|` (поддерживается сложение, вычитание, умножение, деление и функции print(), chr())
- **Комментарии**: `#[ многострочный комментарий ]#`
- **Имена переменных**: `[a-z]+` (только строчные буквы)
